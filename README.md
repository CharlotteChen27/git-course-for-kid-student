# Git 小练习：星星图书馆

这个仓库是用来练习 Git 的。你会一边改一个小故事，一边学习 commit、diff、branch、merge 和 conflict。

## 开始方法

如果还没有把仓库下载到电脑上，可以运行：

```bash
git clone https://github.com/ChestnutLUO/git-course-for-kid-student.git
cd git-course-for-kid-student
```

然后用 GitHub Desktop 打开这个文件夹。

## 你要看哪些文件

- `EXERCISES.md`：所有题目都在这里。
- `workspace/`：你真正要修改的故事文件。
- `CHEATSHEET.md`：忘记概念时可以看一眼。

## 练习规则

- 主要修改 `workspace/` 里的文件。
- 每次提交前先看 diff。
- commit 信息要写清楚，不要只写 `update`。
- 出现 conflict 不代表你做错了，那是练习的一部分。

## 最后你会完成

- 一个有多个保存点的故事。
- 两个不同结局的分支。
- 一次 merge。
- 一次 conflict 解决。
